# مسیر یادگیری توسعه وب

این مسیر شما را از مبتدی تا یک توسعه‌دهنده وب حرفه‌ای راهنمایی می‌کند.

## مقدمه

توسعه وب به دو بخش اصلی تقسیم می‌شود:
- **Frontend**: بخشی که کاربر می‌بیند و با آن تعامل دارد
- **Backend**: منطق سرور و مدیریت داده‌ها

## Frontend Development

### مرحله 1: مبانی (HTML, CSS, JavaScript)

#### HTML
- ساختار صفحات وب
- تگ‌های اصلی و معنایی
- فرم‌ها و ورودی‌ها
- دسترسی‌پذیری (Accessibility)

#### CSS
- استایل‌دهی پایه
- Flexbox و Grid
- Responsive Design
- CSS Variables و Custom Properties
- Animations و Transitions

#### JavaScript
- متغیرها و انواع داده
- توابع و Scope
- DOM Manipulation
- Events
- Async/Await و Promises
- ES6+ Features

### مرحله 2: فریمورک‌های Frontend

#### React
- Components و Props
- State و Lifecycle
- Hooks (useState, useEffect, etc.)
- Routing با React Router
- State Management (Redux, Context API)
- Testing

#### Vue.js
- Vue Components
- Vuex برای State Management
- Vue Router
- Composition API

#### Angular
- Components و Modules
- Services و Dependency Injection
- Routing
- RxJS و Observables

### مرحله 3: ابزارها و تکنولوژی‌های پیشرفته

- **Build Tools**: Webpack, Vite, Parcel
- **CSS Frameworks**: Tailwind CSS, Bootstrap
- **TypeScript**: Type Safety برای JavaScript
- **Testing**: Jest, Cypress, Testing Library
- **PWA**: Progressive Web Apps
