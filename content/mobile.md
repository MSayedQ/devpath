# مسیر یادگیری توسعه موبایل

یادگیری ساخت اپلیکیشن‌های موبایل برای Android و iOS

## مقدمه

توسعه موبایل به سه رویکرد اصلی تقسیم می‌شود:
- **Native Development**: استفاده از زبان‌های اصلی پلتفرم
- **Cross-Platform**: استفاده از فریمورک‌های چندپلتفرمی
- **Hybrid**: ترکیب Web Technologies با Native

## Cross-Platform Development (توصیه شده برای شروع)

### React Native

#### مبانی
- Setup و Environment
- Components و JSX
- State Management
- Navigation (React Navigation)
- Styling
- API Integration

#### مفاهیم پیشرفته
- Native Modules
- Performance Optimization
- Testing
- Publishing به App Stores

### Flutter

#### مبانی
- Dart Language
- Widgets و State Management
- Navigation
- Material Design و Cupertino
- API Integration

#### مفاهیم پیشرفته
- Custom Widgets
- Animations
- Platform Channels
- Testing
- App Deployment

## Native Development

### Android Development

#### Java/Kotlin
- زبان برنامه‌نویسی Kotlin (توصیه شده)
- Android Studio
- Activities و Fragments
- Layouts (XML)
- RecyclerView
- Room Database
- Retrofit برای Networking

#### مفاهیم پیشرفته
- MVVM Architecture
- Dependency Injection (Dagger/Hilt)
- Coroutines
- Jetpack Compose
- Testing (JUnit, Espresso)

### iOS Development

#### Swift
- زبان Swift
- Xcode IDE
- UIKit و SwiftUI
- View Controllers
- Core Data
- URLSession برای Networking

#### مفاهیم پیشرفته
- Combine Framework
- MVVM/MVIP Architecture
- Dependency Injection
- Testing (XCTest)
